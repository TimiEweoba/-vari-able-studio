## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the fullstack development server (Client @ 5000, Server Proxy):
```bash
npm run dev
```

### Database

Push schema changes to your database:
```bash
npm run db:push
```

### Build & Production

Build the production bundle:
```bash
npm run build
```

Start the production server:
```bash
npm start
```
